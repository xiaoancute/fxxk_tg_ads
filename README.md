# Telegram 过滤词

自用 TG 群组/频道过滤词库，适用于 NagramX 等客户端的自动过滤功能。

## 目录结构

```
filters/
├── 懒人包/                              ← 🎯 选一个，完事
│   ├── 基础防护.txt    （赌博+色情）
│   ├── 标准防护.txt    （+诈骗）
│   └── 全面防护.txt    （全部，广告全消失）
│
└── 原来还要自己拼/                              ← 🔧 自己搭
    ├── 赌博/           赌博博彩 + 原子
    ├── 成人/           色情内容 成人用品 + 原子
    ├── 诈骗/           金融诈骗 社会工程诈骗 + 原子
    ├── 广告/           推广引流 电商交易 + 原子
    └── VPN/            VPN硬广 VPN软广 游戏加速器 + 原子
```

每个 `.txt` 文件是一个完整的正则表达式（单行），可直接粘贴到屏蔽规则。

每个 `原来还要自己拼/` 目录下，**粗体标题的文件是「组合」**（合并了多个原子规则，开箱即用），其余是**原子规则**（最细粒度，给极致定制的人用）。

## 懒人包

不用想，选一个：

| 文件 | 包含内容 | 适用场景 |
|------|----------|----------|
| `懒人包/基础防护.txt` | 赌博 + 色情 | 🏠 **推荐默认**，覆盖最烦人的两类 |
| `懒人包/标准防护.txt` | 赌博 + 色情 + 诈骗 | 🛡 再加一层诈骗防护 |
| `懒人包/全面防护.txt` | 全部五大类 | 🧹 广告全消失，可能有误伤 |

## 自由拼 — 各目录说明

### 原来还要自己拼/赌博/

| 文件 | 说明 |
|------|------|
| **`赌博博彩.txt`** | 赌博博彩完整规则（赌场名 + 体育博彩 + 下注模式） |
| `赌场名称.txt` `下注模式.txt` `体育博彩.txt` | 原子规则 |

### 原来还要自己拼/成人/

| 文件 | 说明 |
|------|------|

### 原来还要自己拼/诈骗/

搞投资的用户可以不选 `金融诈骗.txt` 避免误伤财经讨论。

| 文件 | 说明 |
|------|------|
| **`金融诈骗.txt`** | 金融诈骗（投资理财、加密货币、传销、高利贷、洗钱、杀猪盘） |

| `投资诈骗.txt` `加密货币诈骗.txt` `传销庞氏.txt` `高利贷.txt` `银行卡洗钱.txt` `杀猪盘.txt` `刷单诈骗.txt` `钓鱼冒充.txt` `医疗保健品.txt` | 原子规则 |

### 原来还要自己拼/广告/

只想屏蔽硬广可以只选 `推广引流.txt`；🤑 喜欢薅羊毛的用户建议从 `电商交易.txt` 中排除羊毛返利规则。

| 文件 | 说明 |
|------|------|
| **`推广引流.txt`** | 推广引流（假推荐、紧迫话术、价格对比、联系方式导流、互推、链接推广、频道推荐） |
| **`电商交易.txt`** | 电商交易（淘宝口令、电商链接、羊毛返利、代理招募、虚拟资产交易、游戏推广、搜索 Bot） |
| `假推荐.txt` `紧迫感话术.txt` `价格话术.txt` `联系方式导流.txt` `互推引流.txt` `链接推广.txt` `频道推荐.txt` `电商口令.txt` `电商链接.txt` `羊毛返利.txt` `代理会员.txt` `虚拟资产.txt` `游戏推广.txt` `搜索Bot.txt` `返利薅羊毛.txt` | 原子规则 |

> 💡 薅羊毛提示：`电商交易.txt` 内含 `羊毛返利.txt` 的规则（线报群、漏洞单、神价、白菜价、领券等）。主动找羊毛的用户可自行删除文件中对应的 `(?:...)` 片段。

### 原来还要自己拼/VPN/

技术社群建议只选硬广，避免误伤正常讨论。

| 文件 | 说明 |
|------|------|
| **`VPN硬广.txt`** | 硬广告（机场/VPN 品牌名 + 明确的价格/套餐 + 无限流量话术） |
| **`VPN软广.txt`** | 软广告（品质宣传、试用免费、节点订阅 — ⚠️ 技术讨论可能误伤） |
| **`游戏加速器.txt`** | 游戏加速器推广（迅游、UU 等 — 🎮 不属于翻墙，单独拆分） |
| `品牌名称.txt` `套餐方案.txt` `价格数字.txt` `无限流量.txt` `品质宣传.txt` `试用免费.txt` `节点订阅.txt` | 原子规则 |

## 使用方式

在 NagramX 等客户端中将文件内容粘贴到过滤规则输入框即可。每个文件已包含 `(?i)` 不区分大小写标记。
